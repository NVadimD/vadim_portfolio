The Calorie Counting App is a web-based application built using pure JavaScript, HTML, and CSS to help users track their daily calorie intake. This project was developed as a part of my portfolio to showcase my skills in front-end web development.
To Do List
The "ToDoList" project is a web application built with pure JavaScript and utilizing Local Storage to manage tasks and to-do lists. Functional include task and list creation editing and deletion, marking completed tasks and utilizing local storage to save tasks.
Guess and Gallows
